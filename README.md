# data-analytics-portfolio

# Análisis de ventas de Walmart (2010–2012) con Excel y Power Pivot: dashboard, KPIs y reporte analítico.
Proyecto de análisis descriptivo utilizando el dataset público **Walmart Store Sales Forecasting** (Kaggle).

## Objetivo
Evaluar el desempeño de las tiendas Walmart entre febrero 2010 y octubre 2012, analizando:
- ventas por tipo de tienda (A/B/C),
- tiendas de mayor y menor rendimiento,
- estacionalidad por año y trimestre,
- impacto de semanas con feriados,
- distribución del promedio semanal por tienda.

## Herramientas
- Excel + Power Pivot (modelo relacional)
- Tablas dinámicas y segmentadores
- Dashboard interactivo + reporte analítico escrito

## Entregables
- `Walmart_Analisis_Ventas_Gabriel.xlsx`  
  - modelo de datos (train, stores, features)  
  - hoja `Dashboard` con KPIs y visualizaciones  
  - hoja `Análisis-reporte` con interpretación de resultados
- `Walmart_Reporte_Analitico_Gabriel.pdf`  
  - versión en PDF del dashboard y del informe escrito.

## Principales hallazgos
- Las tiendas tipo A representan ~64% de las ventas del período.
- El 22% de las tiendas (Top 10) generan ~39% de las ventas totales.
- Las semanas con feriado presentan en promedio un +7% de ventas frente a semanas normales.
- Fuerte estacionalidad con picos en Q4 y menor desempeño en Q1.
